Quickstart

Cardinal builds a static site from the files in your current directory and starts a webserver to serve the files locally. Cardinal rebuilds the site whenever the source files change. 

go install github.com/tygern/cardinal@v0.0.7
cardinal

Layout

.
├── template.html
├── index.html
├── about.html
├── style
│   └── app.css
└── images
    └── cardinal.png
  • The template.html file defines your site template. Cardinal replaces the <#content/> tag with site content.
  • Any other HTML files define your site's pages. Cardinal applies the template to each HTML file and creates a file with the same name in the build directory.
  • Cardinal also copies all non-HTML files and directories to the `build` directory.
